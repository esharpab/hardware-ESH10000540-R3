# 00_Rituals — Session Openers

Paste the matching prompt into your AI chat to initialize the session correctly.

---

## Routing table

| Situation | Use this ritual |
|---|---|
| Starting work for the day | `START_OF_DAY.md` |
| Mid-session check-in or scope drift | `MID_SESSION_CHECK.md` |
| Ending the day / closing the session | `END_OF_DAY.md` |
| Returning after a break or > 1 day away | `REENTRY_AFTER_BREAK.md` |
| Cold start — no session context at all | `EMERGENCY_BOOT.md` |

All rituals are read-only prompts. They instruct the AI to analyze and report — not to write files.
Writing only happens when you explicitly direct it to.
