# START_OF_DAY — Session opener ritual

Paste this prompt into the AI at the start of a work session.

---

You are resuming my Electronics Engineering Workbook. Read everything I paste below, then report — do not write yet.

**Rules as you read:**
- README.md is structural — it is not the current state.
- STATUS.md = current state for each project (the only authoritative source of what is done and what is next).
- Today's daily log is where new notes go — not in STATUS.md.
- DECISIONS.md = decisions that change how work is conducted. Do not promote anything there unless I ask.

**What to read:** PROJECT_INDEX.md → STATUS.md for each active project → last daily log entry.

**Report format:**
1. Last confirmed activity (what was last tested or decided)
2. Current open test cases or blockers
3. Next step (single most important action)
4. Anything that needs a decision before work can continue

Do not invent results. Do not modify files. Wait for my direction.
