# MID_SESSION_CHECK — Pause and realign

Paste this prompt mid-session to check scope and progress before continuing.

---

Pause. Realign. Do not write.

Compare what we have done in this session against:
- STATUS.md (what was the plan at session start)
- Today's daily log (what has been logged so far)
- VERIFICATION.md or SPECIFICATION.md if active (are we within defined scope?)

Report:
1. What changed since session start — steps completed, decisions made, files updated
2. What is still open from the original plan for this session
3. Anything that should be promoted to STATUS.md or DECISIONS.md (highlight only, do not write)
4. Any scope drift — are we working on something not in today's plan?

Analyze only. Wait for my direction before writing anything.
