# Specification: Sparrow Complete Product R1

## Scope

This document defines the product-level requirements and acceptance criteria for the Sparrow Complete Product R1.
The Sparrow is a multi-channel test instrument assembled from the following sub-assemblies:

| Sub-assembly | ESH | Rev |
|-------------|-----|-----|
| Accordion A2 Base | ESH10000182 | — |
| Sparrow N-Top | ESH10000535 | R3 |
| Sparrow Fixture Electronics PCBA | ESH10000540 | R3 |
| Fixture Link | ESH10000543 | R2 |
| Active Load | ESH10000536 | R2 |
| PoE | ESH10000534 | R4 |

Test tooling:

| Tool | ESH | Rev |
|------|-----|-----|
| Sparrow Test Adapter | ESH10000654 | R0 |

---

## Category Definitions

| Category | Description |
|----------|-------------|
| System | System-level assembly, power-on, and integration requirements |
| Power | System power distribution and regulated output rails |
| Communication | I2C/SPI/UART bus integrity across sub-assemblies |
| Firmware | Firmware loading and MCU responsiveness |
| Signal | End-to-end signal path integrity (analog IO, digital IO, PWM, Tach) |
| Cosmetic | LED indication and visual quality |
